# Query Templates

A growing library of past queries with the strategy that solved them.
Use these for training operators and for offline regression eval.

| File | Round | Query type | Notes |
|---|---|---|---|
| `example-textual-kis.md` | reference | Textual KIS | Template for documenting a query and its solve |
| `example-video-kis.md` | reference | Video KIS | Template covering the no-screenshot constraint |

When you add a new entry, copy a template and rename it to `<year>-<round>-Q<NN>.md`.
