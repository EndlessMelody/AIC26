# Submission Format

The final submission format will be published by the AI Challenge organizers shortly before the preliminary round. The files in this folder document our **best guess** based on the 2022-2025 editions and serve as a stable interface for our submission script.

| File | Purpose |
|---|---|
| `schema.json` | JSON Schema for a single submission payload |
| `example.json` | A valid example submission |
| `csv-format.md` | Notes on the CSV alternative used in some past editions |

Update this folder as soon as the official spec is released.
