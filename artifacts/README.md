# Artifacts

Reusable assets that are **not code** but feed directly into the contest tool.

| Folder | Contents |
|---|---|
| `prompt-library/` | LLM prompt templates: query rewrite, captioning, rerank, Vietnamese ↔ English |
| `query-templates/` | Worked-out example queries from past editions with our solving strategy |
| `submission-format/` | Example submission JSON files and the schema |

These are committed to git (they are small and team-shared).
